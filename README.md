# AI-Time-Tracker

AI-Powered Daily Time Tracker to help users log, track, and analyze their productivity and activities.

---

## **Tech Stack**

- **Frontend:** HTML, CSS, JavaScript  
- **Dev Tools:** Node.js, npm, live-server  
- **Optional Backend/Database:** Firebase (for storing tasks and user data)  
- **AI Tools:** JavaScript-based activity detection, optional AI reminders/suggestions  

---

## **Features**

- User Login and Logout  
- Home Page with navigation links to Dashboard  
- Dashboard Page showing tracked tasks and activity summary  
- Placeholder sections for:  
  - Track Time  
  - Reports  
  - Settings  
- Real-time activity tracking (if implemented)  
- AI-powered suggestions or reminders (if implemented)
